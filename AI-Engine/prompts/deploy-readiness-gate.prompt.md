Run a mandatory deploy/completion gate review for the provided scope.

Required agent checks:
1. Global Architecture Standards Guardian
2. Global Backend Systems Architect and Builder
3. Global Frontend Backend Contract Guardian
4. Global Security Standards Auditor
5. Global Code Quality Gatekeeper
6. Global Integration Test Orchestrator
7. Global Cost Optimization Strategist
8. Global Premium Cross Platform UX Architect
9. Global Cleanup Hygiene Enforcer

Mandatory rules:
- If any required gate is FAIL, overall result is FAIL.
- If any critical area is untested or unknown, overall result is FAIL.
- If any critical flow is dummy or placeholder, overall result is FAIL.
- If secrets exposure risk is unresolved, overall result is FAIL.
- Deployment/completion is blocked until overall result is PASS.

Output format:
1. Scope summary
2. Gate matrix (PASS/FAIL per gate)
3. Critical findings (severity ordered)
4. Test evidence summary (inputs/outputs, APIs, webhooks, integrations, failure paths)
5. Cost and optimization summary
6. Cross-platform UX/layout boundary summary (web, iOS, Android)
7. Final verdict: PASS or FAIL
8. If FAIL: exact blockers and minimal fix plan

Use concise, audit-ready language and include only actionable findings.
